HOTEL CRM CLEAN - STEP 5

Що додано:
- mobile-first сторінка нового бронювання
- пошук гостя по телефону
- автопідстановка даних гостя
- перевірка доступних номерів
- розрахунок вартості
- створення нового гостя або використання існуючого
- створення бронювання
- запис платежів у payments

Перед запуском:
1. Виконай SQL схему з попередніх кроків у Supabase.
2. Заповни room_types і rooms.
3. Створи .env.local на основі .env.local.example
4. Скопіюй файли в проект:
   D:\PROJECTS_HOTEL_CRM\hotel-crm-clean
5. Запусти:
   npm run dev
